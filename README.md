# Java-CapstoneProject

The main synopsis of the project is to create an interactive java project that recommends the movies to the users. Mainly it involves usage of Java classes, Interfaces, Filters. Firstly the set of movies is suggested for the users to rate. Then based on rating of that user, the set of movies are recommended to them with interactive HTML and CSS styling. The whole project idea is the use of Collaborative Filtering. The IDE used for this project is BlueJ environment. After creating the required classes and interfaces, some main class files are compressed to Zip file. And this zip file is uploaded to the website after a prompt, then the project works as required.
